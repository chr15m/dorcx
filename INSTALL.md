Install
-------

 * Install Django-1.4.1
 * Install python-gnupg
 * Check out the source code into a folder e.g. /var/www/YOUR-DOMAIN/dorcx
 * Install one of the Apache configs
 * Restart Apache
 * Have fun!

or try:

$ cd server
$ make gunicorn

and then point your browser at http://localhost:8000/
